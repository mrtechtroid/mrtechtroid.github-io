Basics



1. prompt (string)

This command will allow us to change the default string of path to string.
Like if you type the command prompt Admin: the command prompt will display as follows:

C:\Users\Administrator> prompt admin$G

admin>

admin>

Now you might be wondering what $G stands for. Basically $G stands for > symbol. There are a list of special codes for certain symbols and their values which are as follows:
Special Code Symbol Special Code Symbol
$A & (Ampersand) $B | (pipe)
$C ( (Left parenthesis) $D Current date
$F ) (Right parenthesis) $G > (greater-than sign)
$L < (less-than sign) $N Current drive
$Q = (equal sign) $S (space)
$V Windows version number $_ Carriage return and linefeed
$T Current time $P Current drive and path
$H Backspace (erases previous character) $E Escape code (ASCII code 27)
$$ $ (dollar sign)


2. title (string)

This command will allow us to change the default title of the command prompt to whatever you want.
Like if you type the command title Hello User the command prompt will like make the title say Hello User

C:\Users\Administrator> title hello User



3. /? and help

Adding the suffix /? behind any command will bring up the help section for that particular command.
Adding the prefix help in front of any command will also bring up the help section for that command.

C:\Users\Administrator> prompt /?

Changes the cmd.exe command prompt.

PROMPT [text]
......

C:\Users\Administrator> help prompt

Changes the cmd.exe command prompt.

PROMPT [text]
......


4. tree

This will relay a tree like structure of folders and files in that folder. If we add anything.txt in front then we can save the output in a file.

C:\Users\Administrator> tree

6. shutdown

This command will shut down the computer. If you want to shutdown the computer after a while then you may use the command shutdown -s -t X where X refers to the number of seconds. Well Dont try it now because the computer will shut down!!!

C:\Users\Administrator> shutdown



7. &&

This will allow us to run two or more commands at the same time.
For eg. ping google.com && ipconfig will run the ping command and then ipconfig

C:\Users\Administrator> ping google.com && ipconfig

C:\Users\Administrator> echo We will be discussing Both These Commands Shortly



8. Ctrl + C

This Hotkey will abort any command running in CMD. Like you have turned on the tree command then this might save you and will terminate the program in between.


9. color

This will allow us to change the colour of the CMD. You can use "help color" to find all the list of colours available. Normally cmd runs in white color you can now make it run on lets say green color.

C:\Users\Administrator> color 2

C:\Users\Administrator>

Now Lets Discuss the various Colour Codes..
0 = Black | 8 = Gray
1 = Blue | 9 = Light Blue
2 = Green | A = Light Green
3 = Aqua | B = Light Aqua
4 = Red | C = Light Red
5 = Purple | D = Light Purple
6 = Yellow | E = Light Yellow
7 = White | F = Bright White


10. |clip

You saw that Ctrl+C in Cmd is used for Stopping a command. So what to do if you would like to copy all the result which you get in your clipword. We will use the |clip command. This prefix will send the output of some command to the clipboard directly

C:\Users\Administrator> tree |clip



11. set /a NO1+NO2

You Can Use The Command set /a for calculating basic arithmetic. Instead of ? add any variable like a,b,c,d,.... Note here that for division it gave result as 0 instead of 0.5. This is because it just gives the integer value.

C:\Users\Administrator> set /a 2+4
6

C:\Users\Administrator> set /a 2-4
-2

C:\Users\Administrator> set /a 2*4
8

C:\Users\Administrator> set /a 2/4
0





Intermediate



1. cd

Now You Might be wondering how to change the folders in cmd. Well here you go. So firstly if you want to move a folder Forward then type cd FOLDERNAME. If you want to go back one folder type cd... If you want to go to a different drive then type DRIVELETTER:If the folder name has a blank/space in between then write the path in double inverted commas.

C:\Users\Administrator> cd Documents

C:\Users\Administrator\Documents> cd..

C:\Users\Administrator> F:

F:>cd "I love CMD"

F:\I love CMD>

3. echo

Using this command you can make certain text to be displayed on the Command Prompt.

C:\Users\Administrator> echo Hello World

4. exit

This Command Will Close The Command Prompt or The Batch File(We will talk about it in detail later...).

C:\Users\Administrator> exit



5. time and date

The command time will display the current system time in CMD. It might ask you to set a new time if you just press enter no changes will be made to the time.
The command date will display the current system date in CMD. It might ask you to set a new date if you just press enter no changes will be made to the date.

C:\Users\Administrator> time

The current date is: 12:12:30.30

Enter the new time:

C:\Users\Administrator> date

The current date is: 01-01-2020

Enter the new date: (dd-mm-yy)

8. copy / xcopy

More Commands

1. netsh wlan show profile WiFi-name key=clear

It Lists out the password of the wifi you want to know the name of. Just replace the WiFi-name to your wifi name.

C:\Users\Administrator>netsh wlan show profile test key=clear



2. net user (USER) *

This command will allow to change the password of Windows if we get the computer on bychance and we can even change it. Please Note: that the command prompt needs to be running with Administrator privileges.This can be used in RECOVERY MODE of Windows by using cmd there as the command prompt starts up with Root Administrator Privaliges

C:\Users\Administrator>net user Administrator 123123



3. telnet towel.blinkenlights.nl

This command will start starwars movie in CMD, it works on Telnet protocol which needs to accessed first

C:\Users\Administrator>telnet towel.blinkenlights.nl



4. doskey/history

This command will get the entire history of command prompt you had written in that particular session.

C:\Users\Administrator>doskey/history



5. ipconfig

This will get various items related to networking in cmd

C:\Users\Administrator>ipconfig



6. ping WEBSITE

This will find the ip-address of the website you wrote and some other details.

C:\Users\Administrator>doskey/ping mrtechtroid.github.io



7. tracert

This will trace the whole route and jumps a packet takes to reach its destination.

C:\Users\Administrator>tracert mrtechtroid.github.io



8. wmic product get nam

This command can view all the installed apps in CMD.

C:\Users\Administrator>wmic product get nam



9. sfc/scannow

This command will check the system for any errors

C:\Users\Administrator>doskey/history



10. chkdsk

This command will search for any errors in the disks and drives. Adding /r will check for bad sectors. Adding /f will fix any problems. /scan will scan for any problems.

C:\Users\Administrator>chkdsk



11. driverquery

This will allow us to find all drivers installed in our PC.

C:\Users\Administrator>driverquery
